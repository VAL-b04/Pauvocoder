# Pauvocoder

Programme de **traitement audio** en **Java** permettant de **modifier la hauteur d’un fichier WAV sans en changer la durée**, basé sur une **fréquence donnée**, avec une petite **interface graphique** réalisée en StdDraw

---

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/VAL-b04/Pauvocoder
cd Pauvocoder

# Compiler les fichiers
javac StdDraw.java StdAudio.java Pauvocoder.java

# Lancer le programme
java Pauvocoder <input.wav> <freqScale>
```
